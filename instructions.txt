===================================================================
     AI & Dev News Dashboard 실행 명령어 가이드
===================================================================

본 프로젝트를 정상적으로 실행하기 위한 명령어 정리입니다.
터미널(PowerShell 또는 CMD)을 열고 각 명령어의 디렉토리 경로에 맞춰 실행하십시오.


1. 로컬 웹 서버 실행 (CORS 에러 및 JSON 파일 읽기 실패 해결)
-------------------------------------------------------------------
브라우저에서 index.html을 직접 더블 클릭하여 실행할 경우, 보안 정책(CORS)으로 인해
data/news.json 파일을 가져오지 못합니다. 로컬 웹 서버를 통해 접속해야 합니다.

* 실행 방법:
  블로그 루트 폴더 (d:\15. seminor\blog)에서 실행:
  
  $ npx http-server -p 8080
  
  (또는 제공된 'run.bat' 파일을 마우스로 더블 클릭하여 간편하게 실행할 수 있습니다.)
  
  구동 후 브라우저에서 다음 주소로 접속하십시오:
  ==> http://localhost:8080


2. 뉴스 스크래퍼 수동 실행 (즉시 스크랩 및 번역)
-------------------------------------------------------------------
OpenAI, Google AI, Anthropic, Cursor 채널로부터 뉴스를 즉시 긁어와 한글로 번역한 후
data/news.json 파일을 업데이트합니다.

* 실행 방법:
  scraper 폴더 (d:\15. seminor\blog\scraper)로 이동 후 실행:
  
  $ cd scraper
  $ node index.js --run


3. 매일 아침 9시 자동 스크랩 실행 (스케줄러 활성화)
-------------------------------------------------------------------
백그라운드 스케줄러를 활성화하여, 컴퓨터가 켜져 있는 동안 매일 오전 9시 정각에
자동으로 크롤링과 번역이 수행되도록 합니다.

* 실행 방법:
  scraper 폴더 (d:\15. seminor\blog\scraper)로 이동 후 실행:
  
  $ cd scraper
  $ node index.js
  
  (스케줄러를 종료하려면 터미널에서 Ctrl + C를 입력하십시오.)
